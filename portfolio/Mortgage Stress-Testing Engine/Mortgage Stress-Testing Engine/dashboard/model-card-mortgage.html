Mortgage Stress-Testing Engine — Model Card

MODEL PD (logistic + XGBoost) + discrete-time hazard · VERSION 1.0
OWNER Vaibhav Sethi · DATA Freddie Mac SFLLD (real) · SCENARIOS Fed 2026 DFAST

1 · Purpose & intended use

Estimate mortgage probability of default and project portfolio default and loss over a 13-quarter horizon under the Federal Reserve's 2026 baseline and severely-adverse supervisory scenarios (CCAR/DFAST-style). Intended as a portfolio-analytics / educational demonstration — not a deployed credit-decisioning system and not used to make lending decisions.

2 · Data

  • Source: Freddie Mac Single-Family Loan-Level Dataset (real).
  • Cohorts: 2005–2008 (crisis) + 2017–2019 (COVID) origination vintages — 350,000 loans, 19M loan-month records.
  • Target: default = ever 180+ DPD (delinquency ≥ 6) or credit-event zero-balance code (02/03/09/15); prepayment (01) is a competing risk.
  • Macro covariates: unemployment rate, HPI growth, 30-yr mortgage rate (FRED).

3 · Features

FICO, original LTV / CLTV / DTI, original rate, original UPB, loan purpose, occupancy, property type, units, borrowers, first-time-buyer flag, channel, property state, origination vintage. Sentinel codes (9999 FICO, 999 LTV/DTI) handled explicitly. The hazard track adds loan age and calendar-quarter macro values, plus a mark-to-market (current) LTV.

5 · Explainability

TreeSHAP global importance. Top drivers: FICO, property state, original rate, DTI, LTV/CLTV. Sign check confirmed — correlation of each feature with its SHAP value: FICO −0.97 (lower score → higher risk), OLTV +0.92, OCLTV +0.86, ODTI +0.91, original rate +0.90. All economically correct.

6 · Stress-projection results

As-of book = 2017–2019 cohort (147,609 loans, $35.4B UPB); 13-quarter horizon; discrete-time hazard rolled forward under Fed 2026 macro paths; LGD 30%.

Severely-adverse losses = 5.5× baseline. Peak quarterly default 0.38% (severe) vs 0.05% (baseline). Drivers: unemployment 4.5%→10% and house prices −30% raise the mark-to-market-LTV and unemployment terms in the hazard.

LGD note: the 50k-loan annual sample under-populates final-disposition loss fields (~257 defaults with a clean realized loss), so a documented ~30% mortgage LGD is used rather than an unreliable empirical estimate.

7 · Limitations & failure modes

  • Trained on Freddie conforming loans — does not represent subprime, jumbo, or non-QM books.
  • Freddie data has no protected-class fields; no fair-lending (ECOA) audit is claimed (would require HMDA join — future work).
  • Macro link is reduced-form (captures unemployment/HPI/rate, not credit-spread or CRE shocks).
  • Scenarios are hypothetical regulatory scenarios, not forecasts. The web simulator is illustrative, not a deployed model.

8 · Reproducibility

random_state = 42 throughout; pinned requirements.txt; deterministic loan sampling by hashed Loan Sequence Number.
